Bug Fixes and Quality of Life Improvements

Creating a Separator in MO2
  1. Right-click anywhere in the left pane of MO2 and select Create Separator
  2. Name the separator Bug Fix + QOL

Installation instructions:

  • Main File - YUP - Base Game and All DLC 12.0
If you do not know what these instructions mean, or how to install mods with MO2, read the Mod Installation Advice section from the MO2 page

- Fixes thousands of bugs with both the base game and DLCs

Unofficial Patch NVSE

Installation instructions:

  • Main File - Unofficial Patch NVSE 1.2.16

- Fixes many bugs like YUP, but takes advantage of NVSE and JIP LN

Weapon Mesh Improvement Mod

Installation instructions:

  1. Download the Main File - Weapon Mesh Improvement Mod 1.3 and install it normally through MO2
  2. Once the mod has been installed, double-click on Weapon Mesh Improvement Mod in the left pane of MO2
  3. In the Filetree tab, right-click on WMIMNV.esp and select Hide
The ESP is hidden because the most important fixes from the mod are made through modified meshes, which do not require the ESP. The ESP does contain some of its own fixes, but many of these fixes are already done by the previous two mods. It also has some bugs, causes many conflicts, and includes questionable gameplay-related changes.

- Fixes mesh errors, iron sight alignment, and many other weapon related bugs

lStewieAl's Tweaks Custom INI

Installation instructions:

  • Download the mod using the download button pictured below
  • Finish the install normally through MO2

- My personal INI for Stewie's Tweaks that enables many of the Bug Fixes/QOL Improvements of the mod. Gameplay changes will be covered in the Gameplay section

As these are just my personal preferences, my INI can be exchanged for any other preset INI (such as Axonis's), or you can download the Optional File - Stewie Tweaks INI 2.1 from the Stewie's Tweaks page which has everything disabled, and go through the INI and tweak all the features to your own liking. You will need some form of an INI in order for the mod to fully function.
