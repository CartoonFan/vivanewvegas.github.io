If there are less visual mods in this guide than you were expecting/hoping, read the paragraph at the bottom of the Included Mods section from the Introduction page for why that is.

Creating a Separator in MO2
  1. Right-click anywhere in the left pane of MO2 and select All Mods -> Create Separator
  2. Name the separator Visuals

New Vegas Redesigned 2 Revised

Installation instructions:

  1. Download the Main File - New Vegas Redesigned 2 Revised 1.1 and begin the installation normally through MO2
  2. You will be prompted with a FOMOD installer, in which you should select the Yukichigai Unofficial Patch, JSawyer Ultimate Edition, and Uncut Wasteland Plus NPCs options on the second page
  3. Finish the installation normally through MO2
The JSawyer Ultimate Edition and Uncut Wasteland Plus NPCs options are patches for a mods that will be installed later. The game will not launch properly until you install the mods that the patches are for. If you do not plan on installing either of those mods, you should not select them in the FOMOD.

- Greatly improves the faces of many NPCs without making drastic changes that make vanilla NPCs stick out like a sore thumb

Anniversary Animation Pack

Installation instructions:

  • Main File - Anniversary Anim Pack 1.9b
  • Optional File - Weapon Mesh Improvement Mod Patch 1.7

- Includes three of Hitman's previous animation mods, Subtle Camera Motion, Iron Sights Recoil Animations, and Fire Animation Variants, as well as some new previously unreleased animations

B42 Weapon Inertia

Installation instructions:

  • Main File - B42 Inertia V1.2

- Gives weapons realistic inertia movement

Compatibility Skeleton

Installation instructions:

  • Optional File - NVCS Vanilla Weights 11

- Allows multiple mods that edit the skeleton mesh to work together, required for B42 Weapon Inertia

Improved LOD Noise Texture

Installation instructions:

  • Main File - Improved LOD noise Texture v1.00

- Improves the generic LOD noise texture

Vanilla LOD

Installation instructions:

  • Download the mod using either of the two download buttons pictured below
  • Finish the install normally through MO2

- LOD generated through FNVLODGen for use with vanilla textures that allows for objects to be seen from much farther away

Simple Interior Lighting Overhaul

Installation instructions:

  • Main File - Simple Interior Lighting Overhaul 1.2

- Gives interiors a more atmospheric and immersive feel

Altitude

Installation instructions:

  • Main File - Altitude 2.1
If you do not know what to do at the Extract BSA pop-up, read the Mod Installation Advice section from the MO2 page.

- Vastly improves all the weathers in the game, while still keeping a relatively vanilla-esque feeling

As visual mods are very subjective, not everyone will like Altitude. If you want an alternative, I suggest checking out Nevada Skies. It's a great mod and its fully compatible with the guide if you install it instead of BOTH Altitude and Simple Interior Lighting Overhaul. They can not be used at the same time.

A Little More Lamplight

Installation instructions:

  • Main File - A Little More Lamplight 1.2

- Enhances the vanilla streetlights and adds new ones around Outer Vegas, Freeside, and more
