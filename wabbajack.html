Wabbajack

The Wabbajack Viva New Vegas modlist will allow you to quickly and easily install the same modlist that the full manual guide (VNV Extended included) would result in.

Introduction and Setup

Read the Introduction first, then proceed to the Setup page. Once you're done, keep following this page.

Installation

  1. Download the latest Wabbajack.
  2. Create a new folder anywhere outside of any default Windows folders.
  3. Place the downloaded Wabbajack.exe in this folder, then run it.
  4. Enable Show Unofficial Lists, then locate Viva New Vegas in the gallery and click the download button in the corner.
  5. In Installation Location select an empty folder that is NOT the following:
    • The Steam folder.
    • Any default Windows folders.
    • The game folder.
    • The folder where you put Wabbajack.exe.
  6. Begin the installation.
  7. Accept the Nexus Mods API request.
  8. If you are not a Premium user you will need to manually click download for each mod.
  9. Once complete, the installation will show a green Installation Complete screen.
    • If you see a red Installation Failed screen, please join the ModdingLinked Discord server for support.

Post-Install Steps

  1. Head to the location where you installed the modlist.
  2. Open the Tools folder, then open the 4GB Patcher folder.
  3. Copy the .exe file to the game's Root folder.
  4. Right-click on FNVpatch.exe and select Run as administrator.
  5. A command prompt window will open and should say FalloutNV.exe patched!
  6. Close the command prompt and a file named FalloutNV_backup.exe should appear in the game's Root folder.
  7. In your Installation Location, open Mod Organizer 2.
  8. Expand the Utilities separator, you will see multiple NVTF configurations that are disabled. Read the notes then decide which fits your setup best and enable it.
  9. Now you can play the game through the New Vegas option in the top right:
The Viva New Vegas Wabbajack already comes with all plugins decompressed, this improves performance and mitigates stutter. It also comes with better LOD generated with our guide.

Optional Steps

FPS Limiter

Follow the limiter section of the Performance Guide in order to massively reduce your latency (this is highly recommended because the Wabbajack only comes with VSync as default, which is meant to be used as a limiter which we cannot pre-configure for the user).

Decompressor

BSA Decompressor will give you slightly better performance and faster loading times at a small disk space cost.
  1. In the same Tools folder as before, open the BSA Decompressor folder.
  2. Run FNV BSA Decompressor.exe.
  3. If the game path isn't already there, enter it.
  4. Click Decompress, wait for the process the finish, then exit the program once finished.

DXVK

DXVK is mainly used to improve performance in CPU-limited scenarios, which are the main cause of low FPS in this game.
  1. Read the mod description available here.
  2. Once you know if your GPU supports DXVK and what version to use, enable the correct DXVK for your setup in Mod Organizer, under the Utilities separator.

NVHR

Improves performance by optimizing the game's heap.
  1. Head to the Tools folder, then the NVHR folder.
  2. Take all files and drag them to the game's root folder.
