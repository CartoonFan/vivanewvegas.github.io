Creating a Separator in MO2
  1. Right-click anywhere in the left pane of MO2 and select All Mods -> Create Separator
  2. Name the separator Visuals

Anniversary Animation Pack

Installation instructions:

  • Main File - Anniversary Anim Pack 1.9a
  • Optional File - Weapon Mesh Improvement Mod Patch 1.7

- Includes three of Hitman's previous animation mods, Subtle Camera Motion, Iron Sights Recoil Animations, and Fire Animation Variants, as well as some new previously unreleased animations

Improved LOD Noise Texture

Installation instructions:

  • Main File - Improved LOD noise Texture v1.00

- Improves the generic LOD noise texture

Yellow Goodbye

Installation instructions:

  1. Download the Main File - Yellow Goodbye 1.2.2 manually and begin the installation normally through MO2
  2. You will be prompted with a BAIN installer, in which you should select standard version then OK

- Removes the atmospheric yellow tint, while still keeping a very vanilla feel

Alternative Weather Mods

As weather mods are very subjective, here is a list of the best alternatives to Yellow Goodbye. Clarity is what Yellow Goodbye is based on. It fully removes the tint rather than keeping a little bit of orange. Realistic HDR Lighting goes for a more "realistic" and bright feel, while still staying relatively vanilla-esque. Finally, there is Realistic Wasteland Lighting Redesigned. It features a MCM where you can configure many aspects of it, along with overhauled interior lighting.
